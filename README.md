# Restaurante El Oso

Sitio web oficial del Restaurante El Oso, ubicado en Playa Cuastecomates, Jalisco.  
Aquí encontrarás nuestro menú de mariscos frescos, especialidades y cocteles, además de información de contacto y promociones.

## Tecnologías utilizadas
- HTML5
- CSS3
- Bootstrap 5

## Contenido
- `index.html`: Página principal con menú y secciones.
- `styles.css`: Estilos personalizados.
- `script.js`: Funciones para modales y promociones.
- Carpeta de Imágenes: Fotos de platillos y logotipo.
